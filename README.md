## How To Use

To clone and run this application, you'll need [Git](https://git-scm.com) and [Node.js](https://nodejs.org/en/download/) (which comes with [npm](http://npmjs.com)) installed on your computer.

### Run the project

update `.env` with backend url

```bash
# install packages
yarn

# start
yarn start
```
